## Install

```sh
npm i jb-vid-player
```

## Usage

```jsx
import VidPlayer from "jb-vid-player";
```

## Demo

```jsx
import VidPlayer from "jb-vid-player";

function App() {
  return (
    <>
      <VidPlayer vidSrc="video.mp4" />
    </>
  );
}

export default App;
```

## Prop

- **vidSrc** : Requires URL or file path of the video to be played.
